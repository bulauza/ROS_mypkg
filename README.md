RoboSys homework2(ROS)
===

## 概要
* LEDを点灯させる。  
* LEDは5秒おきに3秒間点灯を繰り返す。  
* LEDが点灯しているかをweb上で確認することが出来る。  
* webでは現在時刻も表示する。

## 使用環境
Raspberry pi3  
OS: ubuntu16.04.5 LTS  

## 実行方法
* LED点灯
  * `roslaunch mypkg time.launch`

* ブラウザで`http://ラズパイのIPアドレス:8000`を閲覧
